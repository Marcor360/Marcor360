## 👋 Sobre mí

Soy **Marco**, desarrollador **Full-Stack freelance** basado en **Querétaro, México**. Construyo aplicaciones web e interfaces **responsive** con **Angular**, **React**, **TypeScript** y **Node.js**, priorizando rendimiento, accesibilidad y código mantenible.

Trabajo de cerca con **marcas de e-commerce mexicanas**: además del desarrollo, diseño estrategias de **SEO** y **GEO (Generative Engine Optimization)** para mejorar su visibilidad tanto en buscadores tradicionales como en motores de IA (ChatGPT, Perplexity, Gemini, Claude).

> Mi objetivo: entregar producto **rápido, medible y bien construido**, no solo código que funcione.

- 🧩 **Frontend** — Angular, React, TypeScript, UI responsive (mobile-first)
- ⚙️ **Backend** — Node.js / Express, APIs REST, MySQL
- 📈 **SEO &amp; GEO** — visibilidad en buscadores y en respuestas de IA
- 🛒 **E-commerce** — integraciones y optimización (incl. Salesforce Commerce Cloud)
- 🧹 Obsesionado con la **arquitectura limpia**, el rendimiento web y el código legible
